# Getting Started with GitHub Pages

GitHub Pages is a fantastic way to host static websites directly from your GitHub repository.

## Key Features

- **Free hosting** - Host your site for free on GitHub's infrastructure
- **Custom domains** - Use your own domain name
- **Markdown support** - Write content in Markdown
- **Jekyll integration** - Built-in support for Jekyll static site generator

## Basic Setup

1. Create a repository
2. Add `_config.yml` for Jekyll configuration
3. Create markdown files for your content
4. Push to GitHub
5. Enable GitHub Pages in repository settings
